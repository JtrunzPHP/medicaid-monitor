What this does: Tracks Medical Loss Ratios (MLR) for Medicaid MCOs across all states using CMS public data. Identifies states where MCO losses signal upcoming mid-year rate adjustments — a leading indicator for Medicaid rate increases that affect provider companies (ENSG, ACHC, ADUS, etc). MLR >90% = rate adjustment likely. MLR >95% = distress.

Data: CMS MLR Summary Reports (data.medicaid.gov) — plan-level MLR filings for 2018-2020.
Parent matching: Plans matched to CNC/MOH/ELV/UNH/CVS via subsidiary name patterns. "UNMATCHED" = smaller/regional MCOs.
Rate adjustment logic: States with weighted MLR >90% flagged. Historic adjusters (OH, TX, GA, KY, IN, FL, VA, PA) scored higher.
Limitations: CMS public data currently covers 2018-2020. Newer data flows through MDCT-MCR (not yet public). Revenue figures in millions.
Investment use: When MCOs lose money in a state → state raises capitation rates → higher revenue for provider companies billing Medicaid in that state.
